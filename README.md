## 🚀 Portfolio Projects

### 📦 [E-Commerce Sales Performance Analysis](https://github.com/aspershupadhyay/ecommerce-sales-analysis)
> **Tools:** SQL · Excel · Power BI  
Analyzed 100K+ transaction records to uncover revenue trends, top-performing product categories, and regional performance gaps. Built an interactive Power BI dashboard tracking MoM sales KPIs.  
**Key insight:** Q4 drives 38% of annual revenue — identified inventory pre-stocking opportunity.

---

### 📉 [Customer Churn Analysis — Telco Dataset](https://github.com/aspershupadhyay/customer-churn-analysis)
> **Tools:** Python · Pandas · Seaborn · Matplotlib  
EDA on 7,000+ telecom customer records to identify churn drivers. Found month-to-month contract holders churn 3× more than annual subscribers. Delivered 3 data-backed retention recommendations.  
**Key insight:** Contract type + tenure are the top 2 churn predictors.

---

### 👥 [HR Analytics Dashboard — Workforce Insights](https://github.com/aspershupadhyay/hr-analytics-dashboard)
> **Tools:** Power BI · DAX · SQL  
Interactive 3-page Power BI dashboard analyzing 22K+ employee records. Built 12 DAX measures including attrition rate, department headcount, and avg tenure. Implemented Row-Level Security.  
**Key insight:** Sales dept has 23% attrition — highest in the org.  
🔗 [View Live Dashboard →](https://app.powerbi.com/YOUR_LINK_HERE)

---

### 🚲 [Seoul Bike-Sharing Demand Analytics](https://github.com/aspershupadhyay/Seoul-Bike-Sharing-Demand-Analytics)
> **Tools:** Python · Pandas · Seaborn  
Cleaned and analyzed 8,760-row weather + temporal dataset. Identified temperature (0.73 correlation) and peak hour (5–6 PM weekdays = 2.1× daily avg) as key demand drivers.  
**Key insight:** Dynamic pricing window recommended for 15°C–25°C range.
